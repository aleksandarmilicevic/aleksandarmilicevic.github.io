### Research Projects

  * [Alloy*](http://alloy.mit.edu/alloy/hola/): a general-purpose
    higher-order relational constraint solver.

  * [aRby](http://people.csail.mit.edu/aleks/arby): an embedding of
    Alloy in Ruby.

  * [Squander](http://people.csail.mit.edu/aleks/squander): a unified
    environment for execution of declarative specification (written in
    first-order relational logic) and imperative Java code.

  * [Jennisys](http://research.microsoft.com/en-us/projects/jennisys):
    a programming language and a synthesis tool from declarative
    first-order specifications to imperative code.

  * [The Alloy Analyzer](http://alloy.mit.edu): an automated model
    finder for a first-order relational specification language.
  
  * [Korat](http://korat.sourceforge.net): a tool for
    bounded-exhaustive generation of test inputs based on complex
    constraints the inputs must satisfy.

  * [Puzzler](http://people.csail.mit.edu/jnear/puzzler): an automated
    solver for natural-language logic puzzles (in the style of
    Einstein's Puzzle).
